RootFi is a unified API provider designed to streamline e-commerce integrations. By simplifying the process of connecting to various E-Commerce platforms, RootFi helps businesses access and manage their customers e-commerce data with ease.

The first step in working with RootFi is to connect an integration - this is the e-commerce platform whose data you want to retrieve and manipulate through RootFi's unified API.

### How do I set up and enable e-commerce integrations?

Before you can begin pulling data via RootFi, it is generally necessary to sign up with the e-commerce software provider of your choice, such as Shopify and BigCommerce.

We have compiled detailed information on how to get set up with these various platforms, which you can find in the pages below. By following these instructions, you can quickly and easily connect with your preferred e-commerce software provider and start utilising the benefits of RootFi.
