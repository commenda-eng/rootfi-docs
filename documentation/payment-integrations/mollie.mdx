To start using our Mollie integration, you need first get the API Key from Mollie.

These steps are outlined in greater detail below.

---

### Step 1 - Get Your API Key

1. Log in to your Mollie dashboard.
2. Go to More > Developers.
   ![](/images/mollie/dashboard.png)
3. You will see your API Keys here. Copy the Live API key or the Test API key.
   ![](/images/mollie/apikey.png)
