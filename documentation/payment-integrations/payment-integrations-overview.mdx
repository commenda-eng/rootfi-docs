RootFi is a unified API provider designed to streamline payment integrations. By simplifying the process of connecting to various payment platforms, RootFi helps businesses access and manage their financial data with ease.

The first step in working with RootFi is to connect an integration - this is the payment platform whose data you want to retrieve and manipulate through RootFi's unified API.

### How do I set up and enable Payment Integrations?

Before you can begin pulling data via RootFi, it is generally necessary to sign up with the payment software provider of your choice, such as Stripe and Razorpay.

We have compiled detailed information on how to get set up with these various platforms, which you can find in the pages below. By following these instructions, you can quickly and easily connect with your preferred payment software provider and start utilising the benefits of RootFi.
