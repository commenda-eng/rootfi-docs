title: "Odoo Accounting"
description: "Learn how to set up our Odoo Accounting integration."
---

In order to get started using your own credentials for our Odoo Accounting integration, your customer will need to first create an API Key for their Odoo application. After that, they will need to enter their Odoo credentials on Integration Labs when prompted during the Odoo integration setup on Integration Labs's Connect Link.

These steps are elaborated upon in detail below.

---

### Part 1: Register Odoo application

1. Go to your Odoo Dashboard.
2. Click on top right corner (on the user name) and select Preferences.

![](/images/bede84b-image.png)

1. Go to Account Security tab and create your first API key.
2. Enter the name of the key and copy the API key generated.

### Part 2: Setting up the integration Using Integration Labs Connect

When your customer is connecting their Odoo account to Integration Labs, they will need to enter their Odoo credentials on Integration Labs's invite link setup.
They will need to fill the following information:

1.  **Domain Name**: The url for your Odoo dashboard. For Example: `Integration Labs.odoo.com`.
2.  **Email Address**: The email with which you will login to your Odoo Dashboard.
3.  **API Key**: The key generated in the above steps.
    ![](/images/odoo_connection.png)
