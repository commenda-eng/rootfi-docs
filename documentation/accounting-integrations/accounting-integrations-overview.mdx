Integration Labs is a unified API provider designed to streamline accounting integrations. By simplifying the process of connecting to various accounting platforms, Integration Labs helps businesses access and manage their financial data with ease.

The first step in working with Integration Labs is to connect an integration - this is the accounting platform whose data you want to retrieve and manipulate through Integration Labs's unified API.

### How do I set up and enable Accounting Integrations?

Before you can begin pulling data via Integration Labs, it is generally necessary to sign up with the accounting software provider of your choice, such as Zoho Books and Xero.

We have compiled detailed information on how to get set up with these various platforms, which you can find in the pages below. By following these instructions, you can quickly and easily connect with your preferred accounting software provider and start utilising the benefits of Integration Labs.
