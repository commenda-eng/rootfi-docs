openapi: openapi get /payments/disputes

[The Disputes Data Model](https://docs.rootfi.dev/api-reference/disputes/overview) is designed to streamline the management and retrieval of disputes information across various payments platforms.

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.rootfi.dev/v3/payments/disputes  \
  --header 'Content-Type: application/json' \
  --header 'api_key: <api_key>' \
  --data '{
  "take": 10,
  "skip": 0,
  "orderBy": {
    "rootfi_created_at": "asc"
  },
  "filter": {
    "status": {
      "eq": "WON"
    }
  }
}'
```

```javascript NodeJS
import { RootFiClient } from "rootfi-api";
const rootfi = new RootFiClient({ apiKey: "<api_key>" });

await rootfi.payments.disputes.list({
  take: 10,
  skip: 0,
  filter: {
    status: { eq: "WON" },
  },
  orderBy: {
    rootfi_id: "asc",
  },
});
```

</RequestExample>
