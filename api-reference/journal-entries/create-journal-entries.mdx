---
openapi: accounting post /accounting/journal_entries
---

The [Journal Entries](https://docs.rootfi.dev/api-reference/journal-entries/overview) are basic records of a company's financial transactions used to record the debits and credits of each transaction.

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.rootfi.dev/accounting/journal_entries \
  --header 'Content-Type: application/json' \
  --header 'api_key: <api-key>' \
  --data '{
  "company_id": 1,
  "data": [
    {
      "document_number": "ROOTFI01",
      "posted_date": "2021-09-28T15:00:00.000Z",
      "description": "Test Invoice",
      "currency_id": "GBP",
      "journal_lines": [
        {
          "account_id": "123456",
          "type": "DEBIT",
          "net_amount": 100,
          "contact_id": "123456",
          "tax_id": "123456",
          "tracking_category_ids": [],
          "description": "Test Item"
        },
        {
          "account_id": "123456",
          "type": "CREDIT",
          "net_amount": 100,
          "contact_id": "123456",
          "tax_id": "123456",
          "tracking_category_ids": [],
          "description": "Test Item"
        }
      ]
    }
  ]
}'
```
