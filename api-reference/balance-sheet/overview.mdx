---
title: "Overview"
description: "A [Balance Sheet](https://www.investopedia.com/terms/b/balancesheet.asp) is a financial report that shows a business's assets, liabilities, and equity at a specific point in time. In an accounting platform, a balance sheet is typically generated automatically based on the financial transactions that have been recorded in the system."
