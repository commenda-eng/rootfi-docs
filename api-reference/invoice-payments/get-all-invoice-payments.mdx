**The Invoice Payments Data Model** streamlines the representation and management of invoice payment information across multiple accounting platforms.

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.rootfi.dev/accounting/invoice_payments/all \
  --header 'Content-Type: application/json' \
  --header 'api_key: <api-key>' \
  --data '{
  "take": 10,
  "skip": 0,
  "orderBy": {
    "amount": "asc"
  },
  "filter": {
    "contact_id": {
      "eq": "1234567890123456789"
    }
  }
}'
```

```javascript NodeJS
import { RootFiClient } from "rootfi-api";

const rootfi = new RootFiClient({
  apiKey: "<api_key>",
});

await rootfi.accounting.invoicePayment.list({
  take: 10,
  skip: 0,
  orderBy: {
    amount: "asc",
  },
  filter: {
    contact_id: { eq: "1234567890123456789" },
  },
});
```
