---
openapi: openapi delete /invite_link/{invite_link_id}
---

**Note:** You can only delete those invite links which is not connected to any company.

<RequestExample>

```bash cURL
curl --request DELETE \
 --url https://api.rootfi.dev/invite_link/{invite_link_id} \
 --header 'api_key: <api-key>'

```

```javascript NodeJS
import { RootFiClient } from "rootfi-api";

const rootfi = new RootFiClient({
  apiKey: "<api_key>",
});
await rootfi.core.inviteLinks.delete("1234567");
```

</RequestExample>
