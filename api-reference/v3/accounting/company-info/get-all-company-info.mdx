**The Company Info Data Model** is designed to simplify the management and retrieval of company-related information across various accounting platforms.

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.rootfi.dev/v3/accounting/company_info  \
  --header 'Content-Type: application/json' \
  --header 'api_key: <api-key>' \
  --data '{
  "take": 10,
  "skip": 0,
  "orderBy": {
    "base_currency": "asc"
  },
  "filter": {
    "name": {
      "eq": "Rootfi Technologies Pvt Ltd"
    }
  }
}'
```

```javascript NodeJS
import { RootFiClient } from "rootfi-api";
const rootfi = new RootFiClient({ apiKey: "<api_key>" });

await rootfi.accounting.companyInfo.list({
  take: 10,
  skip: 0,
  orderBy: {
    base_currency: "asc",
  },
  filter: {
    name: { eq: "Rootfi Technologies Pvt Ltd" },
  },
});
```

</RequestExample>
