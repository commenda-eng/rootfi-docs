Cash flow statements are financial statements that show the inflow and outflow of cash and cash equivalents from a company's operations, investments, and financing activities over a specific period of time.

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.rootfi.dev/v3/accounting/cash_flow_statement  \
  --header 'Content-Type: application/json' \
  --header 'api_key: <api-key>' \
  --data '{
  "take": 10,
  "skip": 0,
  "orderBy": {
    "period_end": "asc"
  },
  "filter": {
    "period": {
      "gt": 4000
    }
  }
}'
```

```javascript NodeJS
import { RootFiClient } from "rootfi-api";
const rootfi = new RootFiClient({ apiKey: "<api_key>" });

await rootfi.accounting.cashflowStatement.list({
  take: 10,
  skip: 0,
  filter: {
    period: { gt: 4000 },
  },
  orderBy: {
    period_end: "asc",
  },
});
```
