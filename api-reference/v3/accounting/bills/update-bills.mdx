A **Bills record** of goods or services purchased from a vendor.

<Info>Only supported for Zoho Books, Quickbooks, Xero</Info>

### Required Permissions

To update bills, you need to have the following permissions:
- `bills.write`
- `accounting.write`

Please ensure your authentication token includes these required permissions.
