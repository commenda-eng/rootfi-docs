---
openapi: accounting post /accounting/purchase_orders/all
---

A **Purchase Order** is a record of goods or services purchased from a vendor

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.rootfi.dev/accounting/purchase_orders/all \
  --header 'Content-Type: application/json' \
  --header 'api_key: <api-key>' \
  --data '{
  "take": 10,
  "skip": 0,
  "orderBy": {
    "total_amount": "asc"
  },
  "filter": {
    "status": {
      "eq": "SUBMITTED"
    }
  }
}'
```

```javascript NodeJS
import { RootFiClient } from "rootfi-api";

const rootfi = new RootFiClient({
  apiKey: "<api_key>",
});

await rootfi.accounting.purchaseOrder.list({
  take: 10,
  skip: 0,
  orderBy: {
    total_amount: "asc",
  },
  filter: {
    status: { eq: "SUBMITTED" },
  },
});
```

</RequestExample>
