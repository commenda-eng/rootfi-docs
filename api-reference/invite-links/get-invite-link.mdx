openapi: unified-api get /invite_link/{invite_link_id}

An **invite link** contains API requests for generating and managing invite links, which can be used to connect end users to their preferred accounting platforms.

<RequestExample>

```bash cURL
curl --request GET \
  --url https://api.rootfi.dev/invite_link/{invite_link_id} \
  --header 'api_key: <api-key>'
```

```javascript NodeJS
import { RootFiClient } from "rootfi-api";

const rootfi = new RootFiClient({
  apiKey: "<api_key>",
});

await rootfi.core.inviteLinks.retrieveById("1234567");
```

</RequestExample>
