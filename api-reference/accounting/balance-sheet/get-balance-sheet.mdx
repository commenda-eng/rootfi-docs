A [balance sheet](https://docs.rootfi.dev/api-reference/balance-sheet/overview) is a financial report that shows a business's assets, liabilities, and equity at a specific point in time. In an accounting platform, a balance sheet is typically generated automatically based on the financial transactions that have been recorded in the system.

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.rootfi.dev/accounting/balance_sheet/all \
  --header 'Content-Type: application/json' \
  --header 'api_key: <api-key>' \
  --data '{
  "take": 10,
  "skip": 0,
  "orderBy": {
    "period_end": "asc"
  },
  "filter": {
    "currency_id": {
      "eq": "5000000000000279512"
    }
  }
}'
```

```javascript NodeJS
import { RootFiClient } from "rootfi-api";

const rootfi = new RootFiClient({
  apiKey: "<api_key>",
});

await rootfi.accounting.balanceSheet.list({
  take: 10,
  skip: 0,
  filter: {
    currency_id: { eq: "5000000000000279512" },
  },
  orderBy: {
    period_end: "asc",
  },
});
```
