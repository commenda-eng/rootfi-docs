**The Bank Accounts Data Model** streamlines access of bank account information across various accounting platforms, providing a consistent and unified representation of bank account data.

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.rootfi.dev/v3/accounting/bank_accounts  \
  --header 'Content-Type: application/json' \
  --header 'api_key: <api-key>' \
  --data '{
  "filter": {
    "account_name": {
      "eq": "Rootfi Test"
    }
  },
  "orderBy": {
    "balance": "asc"
  },
  "take": 10,
  "skip": 0
}'
```

```javascript NodeJS
import { RootFiClient } from "rootfi-api";
const rootfi = new RootFiClient({ apiKey: "<api_key>" });

await rootfi.accounting.bankAccounts.list({
  filter: {
    account_name: { eq: "Rootfi Test" },
  },
  orderBy: {
    balance: "asc",
  },
  take: 10,
  skip: 0,
});
```

</RequestExample>
