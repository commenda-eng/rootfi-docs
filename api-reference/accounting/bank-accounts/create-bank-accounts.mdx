openapi: openapi post /accounting/bank_accounts

**The Bank Accounts Data Model** streamlines access of bank account information across various accounting platforms, providing a consistent and unified representation of bank account data.

<RequestExample>

```bash cURL
curl --request POST \
  --url https://api.rootfi.dev/accounting/bank_accounts \
  --header 'Content-Type: application/json' \
  --header 'api_key: <api-key>' \
  --data '{
  "company_id": 1,
  "data": [
    {
      "institution_name": "Bank Central Asia",
      "account_name": "Rootfi Test",
      "category": "DEPOSIT",
      "account_number": "1234567890",
      "balance": 1000000,
      "currency_id": "IDR"
    }
  ]
}'
```

```javascript NodeJS
import { RootFiClient } from "rootfi-api";

const rootfi = new RootFiClient({
  apiKey: "<api_key>",
});

await rootfi.accounting.bankAccounts.create({
  company_id: 1,
  data: [
    {
      institution_name: "Bank Central Asia",
      account_name: "Rootfi Test",
      category: "DEPOSIT",
      account_number: "1234567890",
      balance: 1000000,
      currency_id: "IDR",
    },
  ],
});
```
