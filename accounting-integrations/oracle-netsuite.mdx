To authenticate NetSuite, the following credentials are required:

- Consumer Key
- Consumer Secret
- Token ID
- Token Secret

This guide will demonstrate how to locate or generate these credentials within NetSuite.

## Install the Bundle

<Steps>
  <Step title="First Step">
    In your NetSuite account, navigate to **Customization** > **SuiteBundler** > **Search & Install Bundles** > **List** or search for Search & Install Bundles in the Search bar.
    ![Untitled](/images/netsuite/Untitled.png)
  </Step>
  <Step title="Second Step">Search for the **Integration Labs Bundle** or bundle ID **489902** using the Keyword box. ![Untitled](/images/netsuite/Untitled%201.png)</Step>
  <Step title="Third Step">Select the bundle and click **Install**. ![Untitled](/images/netsuite/Untitled%202.png)</Step>
  <Step title="Fourth Step">
    The bundle will begin installing and you maybe directed to the Installed Bundles page. It may take up to 5 minutes to install. ![Untitled](/images/netsuite/Untitled%203.png)
  </Step>
</Steps>

## Create a new integration to generate the Consumer Key and Consumer Secret

<Steps>
  <Step title="First Step">
  Navigate to **Setup** > **Integration** > **Manage Integrations** > **New** to create a new integration. ![Untitled](/images/netsuite/Untitled%204.png)
  </Step>
  <Step title="Second Step">
   Create an integration with the below details:
   - Any value can be populated in the **NAME** field
   - The **CONCURRENCY LIMIT** should be left unpopulated
   - Ensure the **REST WEB SERVICES box is checked** under SCOPE
   - The **TOKEN-BASED AUTHENTICATION** and **TBA AUTHORIZATION FLOW** boxes should be checked
   - Fill in the **CALLBACK URL** and **REDIRECT URI** with a value of [[ROOTFI CALLBACK URL](/api-reference/v3/overview/environments#callback-urls)]([ROOTFI CALLBACK URL](/api-reference/v3/overview/environments#callback-urls))

![Untitled](/images/netsuite/Untitled%205.png)

  </Step>
  <Step title="Third Step">
   Save the integration to generate the **Consumer Key** and **Consumer Secret**. **This will only be shown once, so store them in a secure place.**
  ![consumer key and consumer secret screenshot.png](/images/netsuite/consumer_key_and_consumer_secret_screenshot.png)
  </Step>
  <Step title="Fourth Step">
    The bundle will begin installing and you maybe directed to the Installed Bundles page. It may take up to 5 minutes to install. ![Untitled](/images/netsuite/Untitled%203.png)
  </Step>
</Steps>

## Assign a user to the role

<Steps>
<Step title="First Step">
Navigate to **Setup** > **Users/Roles** > **Manage Users** or search **Manage Users** in the Search bar.
![Untitled](/images/netsuite/Untitled%206.png)
</Step>
<Step title="Second Step">
In the Manage Users page, either select a user to assign the Role to.

          <aside>

    ðŸ’¡ Most users use an existing user (often yourself) so that an additional NetSuite seat is not consumed.

          </aside>

![Untitled](/images/netsuite/Untitled%207.png)

</Step>
<Step title="Third Step">
In the user's employee record, click **Edit** and go to the **Access** tab. In your Access tab, go to Roles and select **Integration Labs Role Permission**.

![Untitled](/images/netsuite/Untitled%208.png)

</Step>
</Steps>

## Create an access token to get your Token ID and Secret

<Steps>
<Step title="First Step">
In your **Setup** tab, go to **User Roles** > **Access Tokens** > **New** or search **New Access Token** in the search bar to begin creating a New Access Token.

![Untitled](/images/netsuite/Untitled%209.png)

</Step>
<Step title="Second Step">
Select the Application Name (name of the integration you created in Step 2), User and Role (Integration Labs Role Permission) and click **Save** to generate a new access token.

![Untitled](/images/netsuite/Untitled%2010.png)

</Step>
<Step title="Third Step">
Copy the **Token ID** and **Token Secret** provided. **This will only be shown once, so store them in a secure place.**

![token id and token secret screenshot.png](/images/netsuite/token_id_and_token_secret_screenshot.png)

</Step>
</Steps>
