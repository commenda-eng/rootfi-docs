In order to get started using your own credentials for our Chargebee integration, you will need to first register your Chargebee application. After that, you will need to enter your API key on the Integration Labs dashboard.
These steps are elaborated upon in detail below.

---

### Register your Chargebee application

1. Login in to you Chargebee dashboard and go to 'Settings' --> 'Configure Chargebee'
   ![](/images/chargebee/navigate.png)
2. Navigate to the 'API Keys and Webhooks' section and click on 'API Keys'
   ![](/images/chargebee/apikey-sections.png)
3. Click on '+ Add API Key'.
   ![](/images/chargebee/add-apikey.png)
4. Choose the type of access you want to give to the API key, name the key and click on 'Create Key'.
   ![](/images/chargebee/access.png)
   ![](/images/chargebee/key-name.png)
5. Your API key will be generated and will be displayed on the screen.
   ![](/images/chargebee/keys.png)
