In order to get started using your own api key for our Paddle integration, you will need to copy the api key to the RootFi connect link.

These steps are elaborated upon in detail below.

---

### Part 1: Get the API Key from the Dashboard

1. Log in to your Paddle Dashboard
2. Navigate to 'Developer Tools' -> 'Authentication'.
3. Copy the key from the API Keys section.
   ![](/images/paddle/paddle_api_key.png)

### Part 2: Setting up the integration Using Connect by RootFi

When your customer connects their Paddle account to RootFi, they will need to provide their API Key during the setup process via RootFi' invite link.
1. **API Key**: The key generated in the above steps.
   ![](/images/paddle/paddle_connect_screen.png)
