In order to get started using BigCommerce integration, you will need to enter your BigCommerce credentials in the Integration Labs connect link.

The steps to get your BigCommerce credentials are as follows:

---

### Part 1: Register your BigCommerce application

1. Login to your dashboard and go to 'Settings' -> 'API' -> 'Store-level API accounts'
   ![](/images/bigcommerce/settings.png)
2. Click on '+ Create API account'
   ![](/images/bigcommerce/create_api_account.png)
3. Fill the form data.
   ![](/images/bigcommerce/form_data.png)
4. Add required scopes.
   ![](/images/bigcommerce/scopes.png)
5. Click on save to view your credentials.

### Part 2: Setting up the integration using Connect by Integration Labs

When your customer is connecting their BigCommerce account to Integration Labs, they will need to enter their BigCommerce credentials on Integration Labs's invite link setup.
They will need to fill the following information:

1.  **Client Id** The client id generated in the above steps.
2.  **Client Secret** The client secret generated in the above steps.
3.  **Store Hash**: The hash generated in the above steps.
4.  **Token** The token generated in the above steps.
