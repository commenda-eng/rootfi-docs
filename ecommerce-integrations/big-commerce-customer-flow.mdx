In order to get started using BigCommerce integration, you will need to enter your BigCommerce credentials in the Integration Labs connect link.

The steps to get your BigCommerce credentials are as follows:

---

### Register your BigCommerce application

1. Login to your dashboard and go to 'Settings' -> 'API' -> 'Store-level API accounts'
   ![](/images/bigcommerce/settings.png)
2. Click on '+ Create API account'
   ![](/images/bigcommerce/create_api_account.png)
3. Fill the form data.
   ![](/images/bigcommerce/form_data.png)
4. Add required scopes. Make sure to enable the following permissions to avoid deployment errors:
   - Products: Read/Write
   - Orders: Read/Write
   - Customers: Read/Write
   - Content: Read/Write
   - Settings: Read/Write
   ![](/images/bigcommerce/scopes.png)
5. Click on save to view your credentials.

**Note**: Insufficient permissions will result in deployment errors. Double-check that all required scopes listed above are enabled before proceeding with the integration.
